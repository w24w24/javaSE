# 数组中的最大值

## 1.求数组[2,6,1,77,52,25,7]中的最小值

```javascript
// 求一个数组中的最小值
var arr = [2,6,1,77,52,25,7];
var min = arr[0];
for (var i = 1; i < arr.length; i++) {
    if (arr[i] < min) {
        min = arr[i];
    }
}
console.log('该数组中的最小值是：' + min);
```

## 逻辑：

1. 声明一个保存最小值的变量min
2. 默认最小值可以去数组中的第一个元素
3. 遍历这一个数组，把里面每一个数组和min比较
4. 如果这个元素小于min，就把这个元素保存到min里面，否则继续下一轮比较
