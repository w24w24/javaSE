# 淘宝flexible源码分析---立即执行函数

解析：什么是立即执行函数

对于普通函数来说，从定义到使用，最后都要进行调用，它才可以执行。但是立即执行函数不一样，它不需要调用【因为语法的形式就已经决定了它已经被调用了】，直接可以执行

```javaScript
<script>
    //这是一个普通函数
    function fn() {
        console.log(15);
    }
    fn();//只有在这里调用，它才可以执行

    //这是立即执行函数,立即执行函数有两种写法：
    //(1)这是第一种
    (function(a,b) {
        console.log(a + b);
    })(12,20);
    //(2)这是第二种
    (function(a,b) {
        console.log(a * b)
    }(10,20));
</script>
```

主要作用：创建一个独立的作用域，避免命名冲突
